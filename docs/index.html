Features

🌐

分散WireMock管理

複数のWireMockインスタンスにスタブを一括デプロイ。ワンクリックで全インスタンスを同期できます。

📈

負荷試験に最適化

複数のWireMockインスタンスを並列稼働させ、大規模な負荷試験環境を簡単に構築。

💿

SQLite永続化

外部データベース不要。シンプルなファイルベースの永続化でバックアップも簡単。

🗣

日本語UI

日本語・英語の切り替えに対応。シンプルで直感的なインターフェースでスタブを簡単に作成・管理。

🐳

Dockerサポート

Dockerイメージとして配布。docker compose up で簡単に起動できます。

📚

プロジェクト管理

開発・ステージング・本番など、環境ごとにスタブをプロジェクトとして管理。

Quick Start

# WireMock JP を起動
docker compose up -d

# デモ用WireMockインスタンス付きで起動
docker compose -f docker-compose.yml -f docker-compose.demo.yml up -d

# ブラウザでアクセス
open http://localhost:3000

# デモ用インスタンスのURL（プロジェクト作成後に登録）
http://localhost:8081
http://localhost:8082
